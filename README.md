# -Social-Media-Engagement-Dashboard
## Objective
### Track and analyze content engagement metrics.
 ## Dataset
 * likes
 * comments
 * shares
 * saves
 * reach
 * impressions
 * post_datetime
 * post_hour`
 *  day_of_week
 *  content_category
 *   media_type
 ##  Steps Performed

- **Data Collection:** Used Instagram Analytics dataset (Kaggle) with engagement metrics (likes, comments, shares, reach, etc.)

- **Data Cleaning:** Converted datetime, handled missing values, and ensured data consistency using R

- **Feature Engineering:** Extracted hour, day, and month features; created total engagement metric

- **EDA (RStudio):** Analyzed engagement patterns by time, content category, and reach using visualizations

- **Insights Generation:** Identified best posting time, best day, and top-performing content categories

- **Data Export:** Exported cleaned dataset as CSV for dashboard integration

- **Dashboard (Tableau):** Built interactive dashboard with KPIs, trends, and filters

- **Recommendations:** Suggested optimal posting strategy and content improvements based on analysis

 ## Tool & Library Used
* R studio
* Tebleau
* library(readxl)
* library(dplyr)
* library(lubridate)
* library(ggplot2)
